Mouse support for 10ten Japanese Reader

10ten has finally joined the 1980s with support for using the mouse to control the popup window!

This page explains how it works.

Turning it off

First things first, don't like it? You can turn it off from the settings page.

Even if you turn mouse interaction off, you can still selectively enable it when you need it with the press of a key (see the section on pinning the popup below).

How it works: Getting out of the way

We've tried to make the popup keep out of your way as you browse whilst still being accessible when you want to interact with it. The way it works depends on your settings.

  1. Unless you have configured 10ten Japanese reader to require holding down a key in order to show the popup, it will first show the popup with a dotted outline. In this state it cannot be interacted with.

    After 400ms, the popup will change to have a solid outline and a little arrow pointing to the text you looked up. From here, you can move the mouse across the arrow into the popup to change tabs or select entries to copy.

  2. If you have configured a key like Ctrl to show the popup, the popup will not be interactive until you release that key. After you release it, it will immediately become interactive just like in case 1 above.

Clearing the window

If you find the popup occasionally gets in your way, you can quickly dismiss it by pressing Esc (or alternatively x if you change the key in the settings).

Pinning the popup

If you find it hard to keep your mouse inside the window, or you simply want to keep the popup fixed in place for a while, you can pin it by pressing Ctrl. (If you have configured 10ten to show the popup only when pressing Alt, then the default key to pin the popup will be the same.)

For example, if you use Ctrl to show the popup, once you've found something you want to look into, you can tap Ctrl once more and the popup will stay in place until you click the close button, press Esc, or click outside it.

If you tap Ctrl again, the popup will be unpinned.

Note that you pin the popup even if you disable mouse interaction by pressing the keyboard shortcut.

We hope this makes 10ten Japanese Reader more useful to you. We apologize in advance if it introduces any difficulties. Please let us know on Twitter or GitHub below what we can do to fix it.
